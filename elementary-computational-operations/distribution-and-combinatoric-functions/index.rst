Distribution and Combinatoric Functions
---------------------------------------

AIMMS supports several functions to obtain random numbers from discrete
or continuous distribution, and additionally some combinatoric
functions.

The functions for discrete distributions are:

-  :aimms:func:`Binomial`
-  :aimms:func:`Geometric`
-  :aimms:func:`HyperGeometric`
-  :aimms:func:`NegativeBinomial`
-  :aimms:func:`Poisson`

The functions for continuous distributions are:

-  :aimms:func:`Beta`
-  :aimms:func:`Exponential`
-  :aimms:func:`ExtremeValue`
-  :aimms:func:`Gamma`
-  :aimms:func:`Logistic`
-  :aimms:func:`LogNormal`
-  :aimms:func:`Normal`
-  :aimms:func:`Pareto`
-  :aimms:func:`Triangular`
-  :aimms:func:`Uniform`
-  :aimms:func:`Weibull` 

The following functions that operate on distributions are available:

-  :aimms:func:`DistributionCumulative`
-  :aimms:func:`DistributionDensity`
-  :aimms:func:`DistributionDeviation`
-  :aimms:func:`DistributionInverseCumulative`
-  :aimms:func:`DistributionInverseDensity`
-  :aimms:func:`DistributionKurtosis`
-  :aimms:func:`DistributionMean`
-  :aimms:func:`DistributionSkewness`
-  :aimms:func:`DistributionVariance`

The combinatoric functions are:

-  :aimms:func:`Combination`
-  :aimms:func:`Factorial`
-  :aimms:func:`Permutation`
