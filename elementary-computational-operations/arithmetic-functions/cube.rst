Cube
====

.. code-block:: aimms

    Cube(
        x             ! (input) numerical expression
        )

Arguments
---------

    *x*
        A scalar numerical expression.

Return Value
------------

    The function :aimms:func:`Cube` returns :math:`x^3`.

.. note::

    The function :aimms:func:`Cube` can be used in constraints of nonlinear
    mathematical programs.
