.. _.reslim:

.reslim
=======

Definition
----------

    When specified, the solution process stops after ``.reslim`` seconds.
    When specified it overrides the option ``time_limit``.

Datatype
--------

    The value of the ``.reslim`` suffix is numeric.

.. note::

    -  The suffix ``.optcr`` is initialized to ``NA``. AIMMS considers it
       specified when its value is not equal to ``NA``.
