NLPGetIterationCount
====================

The function :aimms:func:`NLPGetIterationCount` returns the iteration count
associated with the last NLP subproblem solved.

.. code-block:: aimms

    NLPGetIterationCount

Arguments
---------

    *None*

Return Value
------------

    The function :aimms:func:`NLPGetIterationCount` returns the iteration count
    associated with the last NLP subproblem solved.
