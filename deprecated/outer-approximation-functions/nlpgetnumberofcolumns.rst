NLPGetNumberOfColumns
=====================

The function :aimms:func:`NLPGetNumberOfColumns` returns the number of columns in
the last NLP subproblem solved.

.. code-block:: aimms

    NLPGetNumberOfColumns

Arguments
---------

    *None*

Return Value
------------

    The function :aimms:func:`NLPGetNumberOfColumns` returns the number of columns in
    the last NLP subproblem solved.
