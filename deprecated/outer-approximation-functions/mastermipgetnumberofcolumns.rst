MasterMIPGetNumberOfColumns
===========================

The function :aimms:func:`MasterMIPGetNumberOfColumns` returns the number of
columns in the last master MIP problem solved.

.. code-block:: aimms

    MasterMIPGetNumberOfColumns

Arguments
---------

    *None*

Return Value
------------

    The function :aimms:func:`MasterMIPGetNumberOfColumns` returns the number of
    columns in the last master MIP problem solved.
