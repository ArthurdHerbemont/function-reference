MasterMIPGetNumberOfNonZeros
============================

The function :aimms:func:`MasterMIPGetNumberOfNonZeros` returns the number of
nonzeros in the last master MIP problem solved.

.. code-block:: aimms

    MasterMIPGetNumberOfNonZeros

Arguments
---------

    *None*

Return Value
------------

    The function :aimms:func:`MasterMIPGetNumberOfNonZeros` returns the number of
    nonzeros in the last master MIP problem solved.
