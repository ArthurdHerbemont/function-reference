MasterMIPGetSumOfPenalties
==========================

The function :aimms:func:`MasterMIPGetSumOfPenalties` returns the sum of the
penalties in the solution of the last solved master MIP.

.. code-block:: aimms

    MasterMIPGetSumOfPenalties

Arguments
---------

    *None*

Return Value
------------

    The function :aimms:func:`MasterMIPGetSumOfPenalties` returns the sum of the
    penalties in the solution of the last solved master MIP.
