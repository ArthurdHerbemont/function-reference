NLPGetNumberOfRows
==================

The function :aimms:func:`NLPGetNumberOfRows` returns the number of rows in the
last NLP subproblem solved.

.. code-block:: aimms

    NLPGetNumberOfRows

Arguments
---------

    *None*

Return Value
------------

    The function :aimms:func:`NLPGetNumberOfRows` returns the number of rows in the
    last NLP subproblem solved.
