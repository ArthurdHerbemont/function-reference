NLPGetNumberOfNonZeros
======================

The function :aimms:func:`NLPGetNumberOfNonZeros` returns the number of nonzeros
in the last NLP subproblem solved.

.. code-block:: aimms

    NLPGetNumberOfNonZeros

Arguments
---------

    *None*

Return Value
------------

    The function :aimms:func:`NLPGetNumberOfNonZeros` returns the number of nonzeros
    in the last NLP subproblem solved.
