System Settings Related Identifiers
-----------------------------------

The following collection of predefined identifiers contains system-related
information. The contents of these identifiers typically corresponds to data
entered in system dialog boxes, such as the **Solver Configuration** dialog box,
the **User Colors** dialog box and the **User** and **Authorization Level Setup**
dialog boxes.
